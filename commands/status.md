# 状态查看 (Status)

查看项目状态、自动驾驶进度、Agent 状态等。

**数据来源:**
- 阶段状态: `.claude/PROJECT_STATE.json`
- 任务状态: `.claude/TASKS.md`
- 项目规则: `CLAUDE.md`

## 命令

```bash
/project-optimizer:status              # 完整状态
/project-optimizer:status --brief      # 简要状态 (一行)
/project-optimizer:status --phase      # 阶段状态
/project-optimizer:status --tasks      # 任务状态 (读取 TASKS.md)
/project-optimizer:status --log        # 会话日志 (读取 TASKS.md)
```
