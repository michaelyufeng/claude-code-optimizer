# 状态查看 (Status)

查看项目状态、自动驾驶进度、Agent 状态等。

## 命令

```bash
/project-optimizer:status              # 完整状态
/project-optimizer:status --brief      # 简要状态
/project-optimizer:status --phase      # 阶段状态
/project-optimizer:status --tasks      # 任务状态
/project-optimizer:status --memory     # 记忆状态
```
