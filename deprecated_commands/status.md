# 项目状态查询

## 指令

读取 `.claude/PROJECT_STATE.json` 并显示当前状态。

## 输出格式

### 如果项目未初始化：
```
❓ 项目尚未初始化

请先运行初始化命令或告诉 Claude "初始化项目"。
```
